About this website

This is a collection of a few classic algorithms used for different and varied purposes.

It is important to apply the right algorithm just as it is important to pick the right data structure.

For each algorithm, we present a brief description of it, what it's used for, its time and space complexity, an example and a java implementation. You can try out the algorithm running on a server.

Graph Algorithms will attempt to draw a graph image.
